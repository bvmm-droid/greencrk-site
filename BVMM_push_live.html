Everyday Actions
Today
Fast things you might do on any given day. Each one queues a change — tap Go Live above when you're ready to publish.

Mark Closed Today: Shows a "Closed today" banner on the site for just today. Set Today's Special: Banner at top of the site. Clear Banner: Removes today's banner. View Live Site: Opens greencrk.com in a new tab. Undo Last Change: Reverses your most recent edit (before pushing). Bulk Price Increase: Raise all prices in a category by a %.

🧪 Preview Before Live
Push a new version to greencrk.com/preview.html first — review it on your phone, then promote to live when you're happy. Your real site is untouched until you promote.

Upload File: Pick an HTML file Claude gave you — goes to preview URL. Push Changes: Apply your pending price/photo/banner edits to preview URL instead of live. Promote: Replaces live greencrk.com with whatever's on preview.html (auto-saves a rollback first).

💾 Backups & Rollback
Every push automatically creates a backup. Use these if a change went wrong AFTER you pushed it live.

Saves a copy of the current index.html to your phone's Files app. Good insurance before big changes.

Opens a list of timestamped backups from previous pushes. Pick one to restore the site to how it looked then.

🛠 If Something's Wrong
Start here when something feels broken. Run Diagnostics first — it'll usually tell you exactly what's wrong.

Run Diagnostics: Checks 6 things and shows what's broken. Force Reload Tool: Dumps Safari's cache and reloads this page — fixes "button doesn't respond" bugs. Clear Stuck State: If the tool seems frozen mid-push. Setup Wizard: Walks you through tool authentication (rarely needed). Copy GitHub Upload URL: Copies the upload page URL to your clipboard, so you can paste in Safari to drop a new tool version fast. Update Worker: Deploys new Worker code without logging into Cloudflare. Only tap this if someone tells you to.
